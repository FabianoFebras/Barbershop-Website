# English

## Barber Shop Website

A simple website using only front-end concepts,part of the EBAC course **"Javascript Fullstack"**.The website simulates the site of a barber shop the objective was to create it using only bootstrap.

### How does it Work

- The website has 5 sections which are "about",a form that the user can send his name and email for the company to return in the "contact" section that doesn't do anything because the focus is to use bootstrap,a "home" section,a "rating" section that simulates feedback of previous satisfieds customers and a "services" section that tells the user all the services the barber shot can provide.

### Tecnologies

- HTML
- CSS
- JavaScript

### Demo

Open the `home.html` file in your browser to run the project.

# Português

## Site de Barbearia

Um site simples usando apenas conceitos de front-end, parte do curso EBAC **"Javascript Fullstack"**. O site simula o site de uma barbearia e o objetivo era criá-lo usando apenas Bootstrap.

### Como funciona

- O site possui 5 seções: "Sobre", um formulário onde o usuário pode enviar seu nome e e-mail para a empresa retornar, na seção "Contato" (que não tem nenhuma função, pois o foco é usar o Bootstrap), uma seção "Página Inicial", uma seção "Avaliações" que simula o feedback de clientes satisfeitos e uma seção "Serviços" que lista todos os serviços oferecidos pela barbearia.

### Tecnologias

- HTML
- CSS
- JavaScript
